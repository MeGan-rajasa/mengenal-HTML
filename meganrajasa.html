Apa itu HTML

HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. Penanda atau markup ini, nanti akan kita sebut HTML berperan untuk menentukan stuktur konten dan tampilan dari sebuah web. Kalau kita ibaratkan nih.HTML itu seperti batu bata untuk membangun rumah. Batu bata ini dapat disusun, hingga menjadi fondasi dasar. HTML memiliki kelebihan seperti memiliki sumber komunitas yang cukup besar dengan pengguna yang luas. Tidak hanya itu, kelebihan lainnya adalah HTML mudah untuk dipelajari oleh seorang Web Developer pemula. kelebihan lainnya adalah HTML merupakan bahasa pemograman yang gratis atau open source
